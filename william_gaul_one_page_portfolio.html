Project Overview

Designing a wrist-worn, closed-loop peripheral nerve stimulation system that detects pathological tremor using inertial sensors and adaptively delivers electrical stimulation to suppress tremor in real time.

System Architecture
  • IMU sensors capture multi-axis motion data from the wrist
  • Signal processing pipeline extracts tremor-band features
  • Gating logic determines when stimulation is allowed
  • Microcontroller generates biphasic stimulation pulses
  • System designed for future AI-driven parameter optimization
Key Results
  • Reliable detection of tremor-dominant motion in 4–12 Hz band
  • Real-time gating reduces unnecessary stimulation
  • Hardware proof-of-concept designed using low-cost components
